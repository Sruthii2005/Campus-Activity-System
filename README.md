A robust, enterprise-grade full-stack application designed to streamline the lifecycle of campus events—from creation and faculty approval to student registration and certificate generation.

## ✨ Key Features

- 🔐 **Role-Based Access Control (RBAC):** Tailored dashboards for Students, Club Members (Organizers), and Administrators.
- 🎟️ **Event Lifecycle Management:** Streamlined workflow for event creation, editing, and faculty approval.
- 📱 **QR-Code Check-in System:** Club members can mark attendance instantly using student-specific QR codes.
- 📜 **Automated E-Certificates:** Instant, personalized certificate generation for participants upon attendance verification.
- 🏆 **Wall of Fame:** A gamified public leaderboard to highlight and reward top student participation.
- 💳 **Paid Event Support:** Integrated flow for paid registrations with placeholder payment verification.
- 🎨 **Modern Aesthetics:** A premium "Glassmorphism" UI with dark mode, smooth transitions, and responsive layouts.

## 🛠️ Tech Stack

### Backend
- **Core:** Java 17, Spring Boot 4.0.5
- **Security:** Spring Security & JWT (JSON Web Token)
- **Data:** Spring Data JPA with H2 (In-memory Database)
- **Validation:** Spring Boot Starter Validation

### Frontend
- **Framework:** React 19 (Vite)
- **Styling:** Vanilla CSS with Modern Glassmorphism
- **Icons:** Lucide React
- **Routing:** React Router 7
